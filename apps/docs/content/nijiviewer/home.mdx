# ホーム

NijiViewer のトップページです。現在配信中の動画をひと目で確認できます。

## 注目のライブ配信

今まさに配信中の動画を視聴者数の多い順に最大 3 件表示します。
カードをクリックするとそのままページ上で動画を視聴できます。

## お気に入りライバーの紹介（ログイン後）

[お気に入り](/nijiviewer/favorites)に登録しているライバーの中からランダムに最大 4 名をピックアップして表示します。
名前やアイコンをクリックすると[ライバー詳細](/nijiviewer/liver)ページへ移動できます。

## クイックアクセス

- **ライブ動画を見る** → [ライブ動画](/nijiviewer/live-videos)ページへ
- **お気に入りを管理** → [お気に入り](/nijiviewer/favorites)ページへ
