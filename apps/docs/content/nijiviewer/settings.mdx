# 設定

サイドバーに表示する VTuber グループの表示・非表示と並び順を管理するページです。

> **ログインが必要です。**

## 表示中のグループ

チェックが入っているグループがサイドバーに表示されます。

- **ドラッグ＆ドロップ**で表示順を自由に並べ替えられます
- チェックを外すとサイドバーから非表示になります
- 最低 1 つは選択が必要です

## 非表示のグループ

チェックを入れると表示中のグループに追加され、サイドバーに表示されるようになります。

設定はアカウントに紐づいて保存されるため、次回ログイン時も同じ設定が引き継がれます。
