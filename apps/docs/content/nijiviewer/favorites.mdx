# お気に入り

お気に入り登録したライバーのライブ配信・動画をまとめて確認できるページです。

> **ログインが必要です。**

## 配信の確認

お気に入りに登録したライバーの配信・動画が一覧で表示されます。

## お気に入りの管理

ページ右上の **Manage Favorites** ボタンからお気に入りにするライバーを追加・削除できます。

## お気に入りへの登録

[ライバー詳細](/nijiviewer/liver)ページのプロフィール欄にある ❤ ボタンを押すと、そのライバーをお気に入りに追加できます。

お気に入りの設定はアカウントに紐づいて保存されるため、ログインすればどの端末からでも同じ内容を確認できます。
