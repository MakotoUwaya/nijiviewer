# ライブ動画

選択した VTuber グループのライブ配信・動画をまとめて確認できるページです。

## グループの切り替え

サイドバーの **Organization** から見たいグループを選択すると、そのグループの動画一覧に切り替わります。
初期表示はにじさんじです。

主な対応グループ:

- Nijisanji（にじさんじ）
- Hololive（ホロライブ）
- 774inc
- その他多数

## 動画の種類

| 種類 | 説明 |
|------|------|
| **ライブ配信中** | 今まさに配信中。**LIVE** バッジと現在の視聴者数が表示されます |
| **アーカイブ** | 過去の配信録画。総再生回数と動画の長さを確認できます |
| **クリップ** | ファンによる切り抜き動画 |

## 動画の視聴

- カードをクリックするとページ上で動画を再生できます
- ライバーのアイコンをクリックすると[ライバー詳細](/nijiviewer/liver)ページへ移動します
