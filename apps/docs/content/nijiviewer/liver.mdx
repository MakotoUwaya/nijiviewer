# ライバー詳細

ライバーのプロフィールと動画をまとめて確認できるページです。

## プロフィール

- アイコン・バナー画像
- チャンネル名・所属グループ
- YouTube チャンネルへのリンク

## 動画タブ

| タブ | 内容 |
|------|------|
| **Live / Upcoming** | 配信中または配信予定の動画 |
| **Archive** | 過去の配信アーカイブ |
| **Clips** | このライバーに関するクリップ動画 |

## このページへの移動方法

- [ライブ動画](/nijiviewer/live-videos)の動画カードにあるライバーアイコンをクリック
- [ライバー検索](/nijiviewer/liver-search)の検索結果をクリック
- [ホーム](/nijiviewer/home)のお気に入りライバーカードをクリック
